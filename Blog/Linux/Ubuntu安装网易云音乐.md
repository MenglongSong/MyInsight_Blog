## Ubuntu安装网易云音乐

### 下载官网deb文件
：先去官网下载最新版本的网易云http://music.163.com/#/download

2：执行命令：

sudo dpkg -i netease——

3：如果报错了 ，执行以下命令

sudo apt-get -f install

4:然后继续执行：

sudo dpkg -i netease——

done！！
